🚀 Sobre o Projeto
Este desafio técnico foi desenvolvido para consolidar conceitos de JavaScript, transformando uma mecânica simples em uma aplicação robusta com separação de responsabilidades e estados de animação.

🛠️ Tecnologias & Destaques
Vanilla JS: Manipulação de DOM, LocalStorage e lógica baseada em dicionários (Lookup Tables).

UX/UI: Sistema de Dark Mode, animações de "shake" e interface responsiva.

IA de Suporte: Utilização estratégica de IAs para refatoração de código e estudos de padrões de projeto.

Persistência: Placar e preferências salvos via navegador.
